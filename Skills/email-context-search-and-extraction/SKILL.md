# Email Context Search And Extraction

## When to Use
Use this skill when you need to execute workflows related to .

## Procedure
1. Define the target email accounts (e.g., personal@gmail.com, college@iiitnr.edu.in) and the search keywords (e.g., 'deadline', 'registrar', 'international program'). 2. Execute a search across these accounts using the defined keywords. 3. Analyze the returned emails to identify communications matching the criteria. 4. Extract key details (Sender, Subject, Date, Content) from the most relevant communication. 5. Summarize the findings, highlighting critical information found, and prepare the extracted data for external storage.
