# Multi Account Email Retrieval And Export

## When to Use
Use this skill when you need to execute workflows related to .

## Procedure
1. Identify the target email accounts (e.g., personal account and institutional account). 2. Execute a search/retrieval operation for the 25 most recent unread emails from each specified account. 3. Process the retrieved email data, including metadata (ID, Subject, Snippet), from all accounts. 4. Export the compiled data into the user's designated storage location (Obsidian vault) for permanent organization.
